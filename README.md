# Veterinary E-commerce System

Professional project developed

## Project Goal
Develop a comprehensive web application for a local veterinary clinic in Aguascalientes. The system will manage real-time inventory, Point of Sale (POS) for physical store sales, and an online store with automated fiscal invoicing.

## Planned Tech Stack
- **Frontend:** React.js
- **Backend/Database:** Firebase
- **Payments:** Mercado Pago API
- **Invoicing:** Mexican e-invoicing API (Facturama / Factura.com)
